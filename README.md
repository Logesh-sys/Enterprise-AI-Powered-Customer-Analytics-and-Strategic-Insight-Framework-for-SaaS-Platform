# Enterprise-AI-Powered-Customer-Analytics-and-Strategic-Insight-Framework-for-SaaS-Platform
Enterprise AI system for SaaS/CRM analytics that combines classification, regression, and clustering to predict revenue, assess churn risk, and segment customers. It uses feature engineering, SMOTE, XGBoost, SHAP-based explainability, and LLM-driven insights to support strategic business decision-making.
